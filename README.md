# Amira — A Keepsake for Amira Hephzibah 🌸

An interactive, mobile-first “keepsake card deck” — a faith-filled tribute that turns a long letter into 16 swipeable chapters.

**Live:** https://amira-chi.vercel.app

## Deploy

Static single-page site. The live file is `index.html`; friends' video messages live in `videos/`. Hosted on Vercel — pushes to `main` auto-redeploy.
